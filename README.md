# RFC-bit

RFC-bit is a command-line tool that fetches RFCs, generates human-friendly explanations with animated visualizations, and creates language-specific implementation guides with code skeletons.

## Features

- **RFC Explanations** - Get clear, beginner-friendly explanations of complex protocols
- **Animated Visualizations** - Auto-generated Manim animations showing protocol flows
- **Implementation Guides** - Language-specific code skeletons (Rust, Python, Go)
- **Two Scope Levels** - Minimalor Practical
- **AI-Powered** - Uses Google Gemini to understand and explain specifications
